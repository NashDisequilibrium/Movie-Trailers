Instructions for running application. 

1. Check that you have the following files. a) .gitattributes b) .gitignore c) entertainment_centre.py c) fresh_tomatoes.html d) fresh_tomatoes.py e) fresh_tomatoes.pyc f) media.py and g) media.pyc

2. In order to run the application, ensure that you have Python (preferably 2.7.9) downloaded

3. Run the file entertainment_centre.py 

4. The file entertainment_centre.py will call the other files, including the html, so that the program is viewable and useable in your browser. 

5. Enjoy watching the trailers for the movies
